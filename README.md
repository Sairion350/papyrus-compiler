
## Compiler

### Prerequisites

* [V compiler](https://github.com/vlang/v/releases)

### Building

1. ```build prod.bat```

### Usage

compile:

```papyrus -compile -input "...\input-dir" -output "...\output-dir"```

```papyrus -compile -nocache -input "...\input-dir" -output "...\output-dir"```

read/print pex file:

```papyrus -read "...\ABCD.pex"```
